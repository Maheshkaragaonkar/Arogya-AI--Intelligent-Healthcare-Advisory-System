# 🏥 Arogya AI - Intelligent Healthcare Advisory System

## 📌 Overview

Arogya AI is an intelligent healthcare web application designed to provide **real-time medical assistance** using Artificial Intelligence and Large Language Models (LLMs).
It helps users get instant health guidance, disease prediction, treatment plans, and medical report analysis.

---

## 🚀 Features

* 🤖 **AI Chatbot** – Real-time medical consultation
* 🩺 **Disease Prediction** – Predict diseases based on symptoms
* 💊 **Treatment Plan Generator** – Personalized medication & lifestyle advice
* 🌿 **Alternative Medicine** – Ayurveda, herbal remedies, yoga suggestions
* 📄 **Medical Report Analysis** – Upload PDF reports and get insights
* 🏥 **Hospital Recommendation** – Suggest nearby hospitals
* 📊 **User Profile-Based Responses** – Personalized AI output

## ▶️ How to Run the Project

```

```

```

```

```

```


1. User registers & inputs health details
2. Data stored in session
3. User selects module (chatbot, prediction, etc.)
4. Input sent to backend
5. AI processes and returns response
